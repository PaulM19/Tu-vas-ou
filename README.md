# Tu pars où ?

Plateforme de mise en relation des étudiants emlyon partant en échange académique.

## Stack
- React 18 + Vite
- Hébergement : Vercel

## Lancer en local

```bash
npm install
npm run dev
```

## Déployer sur Vercel

1. Pusher ce repo sur GitHub
2. Aller sur [vercel.com](https://vercel.com)
3. "New Project" → importer le repo GitHub
4. Vercel détecte Vite automatiquement → cliquer "Deploy"

L'app est en ligne sur `tu-pars-ou.vercel.app` en 2 minutes.
